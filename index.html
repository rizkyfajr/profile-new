Tentang Saya

Hai, saya Rizky Fajar Anugrah, seorang Web Developer yang bersemangat dengan pengalaman langsung dalam membangun, menyesuaikan, dan memelihara aplikasi internal yang mendukung operasi bisnis. Saya berspesialisasi dalam mengembangkan sistem web yang ramah pengguna, efisien, dan dapat diskalakan menggunakan teknologi seperti PHP, Laravel, Vue.js, dan MySQL.

Saya senang memecahkan masalah dunia nyata melalui kode dan terus berusaha untuk meningkatkan keterampilan teknis dan soft skill saya. Minat saya terutama terletak pada pengembangan backend, integrasi sistem, dan optimisasi kinerja.

Saat ini, saya bekerja di Kimia Farma sebagai Web Developer, mengelola dan meningkatkan sistem internal terkait proses HR, gudang, dan manajemen dokumen. Mari terhubung! Saya terbuka untuk peluang baru, kolaborasi, dan diskusi teknologi!
